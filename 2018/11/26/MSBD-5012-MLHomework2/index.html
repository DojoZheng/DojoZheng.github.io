Question 4

Answer Q4.1

The curve (ii) represents the training error. Because it is easy for classifiers to fit for a few training data points correctly, but harder to fit for many training data points. So as the training set size is becoming bigger and bigger, the training error increases as well.

Answer Q4.2

What does the gap between the two curves represent?

The gap between the two curves represent the generalization gap. If the gap can not converge, it means that the classifiers can not generalize well for the test data set. Othervise, it means that the classifiers or the models have low variance and bias.

How does the gap depend on the sample size?

The following picture shows the VC Theorem:

As we can see from the VC Therorem Part 1, the larger the sample size of m, the samller the value of the right formula. Thus, $|\epsilon(h)-\hat{\epsilon}(h)|$ will be samller which leads to a smaller gap. Othervise, if the sample size is very small, the gap will be very big. Hence, more data implies better performance of learning algorithms.

How does the gap depend on the model complexity?

  1. From the VC Theorem, we can see that d=VC(H) represent the model complexity and d has an opposite effect on the gap compared with m. The larger the d, the larger the generalization gap. If d is smaller, then the gap will be smaller as well.
